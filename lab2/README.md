# Лабораторная 2

Построение СКНФ и СДНФ на основании таблиц истинности + расширенные операции над булевыми функциями.

## Что уже есть в каркасе

- парсер логических выражений для операций `!`, `&`, `|`, `->`, `~`
- поддержка переменных `a..e`
- построение таблицы истинности
- построение СДНФ/СКНФ
- числовые формы для СДНФ/СКНФ
- индексная форма функции
- проверка классов Поста (`T0`, `T1`, `S`, `M`, `L`)
- построение полинома Жегалкина
- поиск фиктивных переменных
- булевы производные: частные и смешанные (порядок 1..4)
- минимизация:
  - расчетный метод (со стадиями склеивания)
  - расчетно-табличный (со стадиями склеивания и таблицей покрытия)
  - табличный (карта Карно для 1..4 переменных + визуализация карты)

## Структура

- `main.py` — CLI вход
- `src/core/` — базовый уровень: AST, парсер, вычисление, таблица истинности
- `src/analysis/` — аналитика функции: формы, Пост, Жегалкин, производные, отчет, общий pipeline
- `src/minimization/` — минимизация и промежуточные структуры
- `tests/` — базовые тесты

## Запуск

Из папки `lab2`:

```bash
python main.py
```

или сразу с выражением:

```bash
python main.py "!(a & b) -> (c ~ d)"
```

## Примечания по доработке

- В `src/minimization/karnaugh.py` можно заменить текущий выбор групп на полноценный алгоритм группировок карты Карно.
- В `src/minimization/qm.py` при необходимости можно заменить жадный выбор покрытия на строгий (например, через метод Петрика).
